# CRUD_cinema
Курсовой проект, написанный на 3 курсе СПбГЭТУ (ЛЭТИ). Представляет из себя приложение, работающее с простой БД в MySQL. 
Для написания были использованы Hibernate и Spring Framework.

Честно признаюсь, в проекте была пара неотловленных багов, до которых у меня так и не дошли руки, но оснойвной функционал был реализован нормально.
